# SISSO hardness
Datasets for the prediction of hardness using SISSO 

train.dat - data file with materials properties used for training

Different files correspond to datasets for borides, nitrides, carbides, layered compounds, and all (data_Hpredicted_6.csv)

Using this data please cite the following paper
[C. Tantardini, H.A. Zakaryan, Z.-K. Han, T. Altalhi, S.V. Levchenko, A.G. Kvashnin, B.I. Yakobson, Material hardness descriptor derived by symbolic regression, J. Comp. Sci., 82, 102402 (2024)](https://www.sciencedirect.com/science/article/pii/S1877750324001959)
